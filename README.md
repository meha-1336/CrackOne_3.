📚 Narrative Nook – Design System & Component Library

📌 Overview

Narrative Nook is a UI/UX Design System created for a bookstore application as part of an internship task.
The project focuses on building consistent, reusable components and defining a clear visual style for better user experience.

🎯 Objectives
Maintain design consistency across screens
Build a scalable component library
Improve usability and readability
Support efficient design-to-development workflow

✨ Features
🎨 Defined color palette and typography
🧩 Reusable UI components
🔄 Multiple states (hover, active, disabled)
📱 Clean and minimal layout
⚡ User-friendly interface

🧩 Components
Buttons (Primary / Secondary / Disabled)
Input Fields
Book Cards
Navigation Bar

💡 Future Improvements
Add book images
Implement search and filters
Add animations
Improve accessibility
